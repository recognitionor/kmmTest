# kmmTest

스터디 개요
- 코틀린 멀티 플랫폼 스타디
- 일주일 1회정도 한시간 가량 카페 또는 회사에서

스터디 목적
- 코틀린 멀티 플랫폼 기반으로 클린 아키텍쳐 기반의 양 OS 앱을 만들어보자

준비물 
xcode, android studio 

스터디 순서
1. 환경 세팅 및 최초 프로젝트 푸시
2. 컴포즈로 간단 UI 구현
3. 커몬 리소스에서 리소스 불러오기와 expect/actual 의 이해
4. 이미지 불러오기 와 KTOR 연동
5. 클린 아키텍쳐에 대한 이해
6. 구조에 대한 방향 논의
8. 모듈과 sdk 로 빌드
9. 앱개발
10. 앱 배포
